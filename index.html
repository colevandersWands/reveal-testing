HTML

What is HTML?

  • Hypertext Markup Language
  • HTML is a Language, as it has code-words and syntax like other languages
  • HTML files are created with the extenstsion .html (index.html)
  • HTML uses tags. For example " < " ">" to open a tag and " < / " ">" to close a tag

What is an element in HTML?

An element or element tag is a/an ...

  1. <title >
  2. <header >
  3. <p >
  4. <body >
  5. etc...

Web pages are formed by many types of elements. Linked here is a list of element tags. Practice using the tags in your Code Pen to test them out.

Most of your coding will take place in between an " element". The most common elements you will see are "head" and "body". Majority of your code will take place in the " < body >" tag
